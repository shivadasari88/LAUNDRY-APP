
# Laundry Management Application

A full-stack web application to manage laundry services with role-based access for
Customers, Service Providers, and Admins.

### Phase 1
- Landing Page
- Username-based Registration (No email, No JWT)
- Role-based Login (Customer / Provider / Admin)
- Role-based Dashboards
- Frontend: React (Vite)
- Backend: Spring Boot
- Database: MySQL
